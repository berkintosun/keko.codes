Development is hard. Don't you ever find yourself, thinking about the next big thing? Don't worry, It does happen to everyone

You come up with an amazing idea. You cannot stop thinking about it and fell in love with the idea until actual work comes to place. Humans cannot keep up their motivation all the time. KekoBuddy will solve this problem for you!

You imagine the new big web app? Good, but you need to design the interface and the database structure, code the frontend and the backend. OH, don't forget about the security and devops!

Or a fancy machine learning solution? Nice one! But how are you planning to collect the dataset? You need to build a scrapper, design a database for the data, analyze the data like a data scientist would do and then you can start working on hypertuning those parameters! Ohh don't forget the interface. You think people would use a CLI solution?

KekoBuddy is here to save you from all this and focus on the part you enjoy. In this community, you will find like-minded people who enjoys creating solutions! Everyone have a speciality that they want to focus and KekoBuddy is bringing T shaped people together and matches them to keep the motivation up all times!
